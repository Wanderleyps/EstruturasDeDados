# Estrutura de Dados e Introdução ao Python

Este repositório contém códigos e materiais utilizados na disciplina de Estrutura de Dados do curso de Licenciatura em Informática no IFRN (Instituto Federal de Educação, Ciência e Tecnologia do Rio Grande do Norte). 

## Objetivo
O objetivo principal deste repositório é proporcionar aos estudantes uma compreensão mais aprofundada das estruturas de dados mais comuns, permitindo uma melhor compreensão de algoritmos e lógica de programação. Além disso, fornece uma introdução ao Python, uma linguagem amplamente utilizada em diversas áreas da computação.

## Conteúdo
- Estruturas de dados: listas, pilhas, filas, árvores, grafos, entre outras.
- Códigos-fonte dos algoritmos e estruturas implementadas em Python.
- Materiais didáticos e explicativos.
- Exemplos práticos para compreensão e aplicação dos conceitos aprendidos em aula.

## Organização
O repositório está organizado por estruturas de dados específicas, permitindo uma fácil navegação e referência para os códigos relacionados. Os códigos estão devidamente documentados para facilitar o entendimento.

## Contribuições
Contribuições são bem-vindas! Se você é estudante ou entusiasta e deseja contribuir com códigos, correções ou materiais didáticos, fique à vontade para fazer um *fork* deste repositório e enviar suas sugestões por meio de *pull requests*.

Esperamos que este repositório seja útil para aprimorar seus conhecimentos em estruturas de dados e em programação em Python. Bom estudo!
